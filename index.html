Process Digital Twin for Smart Manufacturing

To design and implement a data-driven Process Digital Twin framework capable of reconstructing, analyzing, and predicting manufacturing system performance using regression-based machine learning models.

Problem Statement

It is important to acknowledge that the current study is based on static experimental data and utilizes regression-based machine learning models to predict critical manufacturing parameters such as surface roughness and power consumption. While effective for controlled experimental environments, this approach does not capture temporal dependencies inherent in dynamic production systems. A structured Process Digital Twin is required to bridge this gap by enabling systematic reconstruction of system behavior, predictive analysis, and future integration with real-time adaptive monitoring systems.

Literature Review

Recent advancements in Industry 4.0 highlight the integration of Digital Twin (DT) technology with machine learning for predictive maintenance and smart manufacturing. Studies demonstrate the application of data-driven DT frameworks in CNC machining operations, where regression and ensemble-based ML models such as Random Forest and XGBoost are used to predict surface roughness and power consumption. Research indicates that ensemble learning techniques outperform traditional regression approaches in non-linear manufacturing processes due to their robustness and variance reduction capabilities. However, most existing frameworks operate on static experimental datasets and lack real-time temporal modeling. This project builds upon these foundations by implementing a structured regression-based Process Digital Twin prototype capable of predictive modeling, comparative algorithm evaluation, and performance metric validation using R², MAE, RMSE, and MAPE metrics.

Research Gap

Existing implementations do not sufficiently incorporate adaptive time-series learning or real-time industrial validation. This project establishes a foundational predictive twin architecture while identifying the need for temporal and real-time extensions.

Results & Analysis

Random Forest achieved highest accuracy for surface roughness prediction (R² ≈ 94%). XGBoost achieved highest accuracy for power consumption prediction (R² ≈ 98%). Results demonstrate effectiveness of ensemble learning within regression-based digital twin modeling.
