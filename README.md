## Overview
This project implements a prototype of a Process Digital Twin using event-log data.

## Objective
To reconstruct and track the time-evolving state of business process instances.

## Components
- Synthetic event log generation
- Process reconstruction logic
- Digital Twin state computation
- Duration and delay analysis

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Generate synthetic data:
   python generate_data.py

3. Run Digital Twin:
   python digital_twin.py

## Future Scope
- Real-time data ingestion
- Time-series database integration
- Predictive AI layer
